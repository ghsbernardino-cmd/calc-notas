# Calculadora de Médias e Situação do Aluno

Um projeto em Python desenvolvido com foco em testes automatizados utilizando a biblioteca **pytest**. 

## Funcionalidades

- Validação de notas (valores permitidos entre 0.0 e 10.0).
- Cálculo de média ponderada (com peso dobrado na terceira nota).
- Classificação automatizada da situação do aluno (`aprovado`, `recuperacao` ou `reprovado`).

## Tecnologias Utilizadas

- **Python** 
- **Pytest** 
- **Git & GitHub**
